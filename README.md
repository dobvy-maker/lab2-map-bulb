This Android application implements two classes: Lamp and Bulb.

The application simulates lamp behavior including:
- turning the lamp on and off
- changing light intensity
- bulb burning when intensity exceeds the limit
- replacing the bulb
- testing required scenarios inside the Android application

The application displays test results directly on the screen.
